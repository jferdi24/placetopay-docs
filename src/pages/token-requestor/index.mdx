# ¿Qué es Token Requestor?

Token Requestor es una solución API que proporciona la capacidad de gestionar de manera
segura la tokenización de tarjetas de pago de diversas marcas (Visa, Mastercard).

El sistema permite la conversión de datos de tarjetas de pago en tokens seguros,
lo que reduce el riesgo de exposición de información sensible y mejora la experiencia
de usuario al evitar la necesidad de almacenar o transmitir datos sensibles de las tarjetas.

La plataforma está diseñada para integrarse con comercios, proveedores de pagos y redes de tarjetas de forma segura y eficiente.


# Objetivo Principal:

Reducir los riesgos asociados al manejo de información sensible mediante la tokenización de tarjetas,
garantizando así una experiencia de pago más segura y confiable tanto para los comercios como para los consumidores.
