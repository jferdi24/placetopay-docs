This web service and its architecture allow any application to connect to Placetopay in order to consume Microsites invoice services.

To use this web service, your application must support a programming language capable of communicating with a REST-type service.

The integration method includes the basic operations for creating, retrieving, settling, and removing invoices.

Below, each endpoint will be described along with the required request data, the resulting response data, and examples of requests and responses for each case.
