# Remover factura

Este endpoint permite **eliminar definitivamente** una factura.

<Note type='warning'> La eliminación **no puede ser revertida**. </Note>

La operación requiere el **id único de la factura**, generado por el sistema al momento de su creación.
La respuesta devuelve la información completa de la factura **tal como estaba registrada antes de ser eliminada**.
