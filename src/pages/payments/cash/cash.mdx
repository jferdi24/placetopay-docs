## Flujo de pago 

1.	Durante el proceso de pago, el cliente recibe un documento PDF con los detalles e instrucciones para completar la transacción.
2.	El cliente debe presentar el documento en un punto de pago autorizado y realizar el pago en efectivo.	
3.	Una vez confirmado el pago, la transacción se procesa automáticamente, y tanto el comercio como el usuario reciben una notificación de la operación completada.

## Integraciones

- **Webcheckout:** Este medio de pago está disponible y listo para usarse a través de [Webcheckout](/checkout). No necesitas preocuparte por verificaciones o validaciones adicionales, ya que PlacetoPay gestiona todo el proceso de pago de manera segura y automatizada. Solo debes de tener en cuenta el tiempo de expiración.

- **Integración API:** Si deseas incorporar este medio de pago directamente en tu plataforma, puedes hacerlo mediante nuestra API. Contacta a tu asesor para más información.

## Restricciones

- Dado que este método de pago requiere la interacción física del usuario, en integraciones mediante Webcheckout es obligatorio que la fecha de expiración sea mayor a 30 minutos.. Consulta más detalles en [WebCheckout - Fecha de expiración](/checkout/expiration).

- El tiempo de expiración aplica para todas las sesiones de pago, independientemente del medio de pago.

## Recomendaciones de integración

- Cuando se integra un comercio con la plataforma de pagos mediante Webcheckout, es posible controlar y personalizar los tiempos de expiración de las transacciones según el medio de pago elegido por el usuario (como Visa, MasterCard, AMEX, PSE o efectivo).

Esto se logra utilizando el parámetro [paymentMethod](/checkout/payment-methods) en la solicitud [Create Request](/checkout/api/reference/session#create-a-session), donde se indica:

- Qué medios de pago estarán disponibles para el usuario.

- Qué tiempo de expiración se aplicará a cada uno.

💡 Recomendación: Antes de redirigir al usuario a la pasarela de pagos, se sugiere mostrarle las opciones de pago en el sitio del comercio y definir el tiempo de expiración adecuado para cada una. Por ejemplo:

Medios como tarjetas o PSE pueden tener expiraciones cortas entre 6 minutos y 30 minutos.

Pagos en efectivo requieren tiempos más extensos, se sugiere un tiempo máximo de 4 horas.

Así se evita que un usuario que elige un medio diferente al efectivo deba esperar a que expire el tiempo asignado innecesariamente para volver a intentarlo.
