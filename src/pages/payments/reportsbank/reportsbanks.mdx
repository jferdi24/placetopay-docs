# Reportes de cierre

 **Estos reportes contienen:**

-	Archivos automatizados (logfile.txt y lockbox.txt).
-	Reportes operacionales en formato PDF.
-	Los reportes se distribuyen por SFTP o a través del Portal Web Evertec. 
-	También están disponibles dinámicamente en el Dashboard de Predictive.
-	Los comercios utilizan estos reportes para conciliar los registros de sus transacciones, ya sea de forma automatizada o manual.

## Reportes de Conciliación Bancaria

Al finalizar la reconciliación interna se generan unos reportes de conciliación en relación con la información transaccional equivalente al depósito bancario.

Como parte de este proceso se generan tres reportes de conciliación para el comerciante:

| Reporte	| Descripción	 | Formato | 
|---------| -------------------------- | -------------------------------------------------------     |
| Lockbox	| Incluye transacciones exitosas y devoluciones de ACH solamente. |	Flat File (.txt) |
| LogFile	| Incluye las transacciones exitosas y fallidas solamente. | Flat File (.txt) |
| PDF     |	Incluye las transacciones exitosas, fallidas y devoluciones de ACH.	| PDF File (.pdf) |

Estos archivos generados por el cierre diario son enviados al servidor de Evertec [Portal de Evertec](/payments/reportsbank/reportsbanks#portal-web-evertec) y al servidor de preferencia del Comercio por medio de un protocolo de seguridad para transferencia de archivos.

Puedes visualizar un ejemplo de los tipos de reportes en el siguiente apartado [Ejemplos de reportes](/payments/reportsbank/examplereportsbanks)

## SFTP (Secure File Transfer Protocol)

Es un protocolo seguro para transferir archivos entre computadoras a través de una red, utilizando cifrado mediante SSH (Secure Shell).
Protege los datos durante la transmisión, requiere autenticación segura y permite realizar operaciones como subir, descargar, renombrar o eliminar archivos en servidores remotos.  


## Datos necesarios para completar en el formulario para configurar SFTP

1.	Credenciales
     -	Usuario ID 
     -	Contraseña 
2.	URL / IP 
3.	Servidor destino
4.	Transmission Security Option
      -	PGP Key (Si o No) | Nota: Si Aplica
     -	SSH Key (Si o No) | Nota: Si Aplica

## ¿Cómo solicitar los reportes de conciliación bancaria de su Comercio?

Para solicitar los reportes de conciliación bancaria correspondientes a su Comercio, le recomendamos comunicarse directamente con su oficial de relación.
El banco adquirente al que pertenece será el encargado de proporcionarle dicha información.

## Cierre del Comercio

Este es el momento en que un Comercio finaliza su jornada de ventas y realiza un corte de las transacciones procesadas durante ese período. 
Define qué transacciones serán incluidas en el procesamiento y conciliación del día.

## Cierre de Red Procesadora

Diariamente, como parte del flujo transaccional la red procesadora realiza un cierre para las transacciones realizadas por el comerciante. 
Este cierre se realiza en los días hábiles del procesamiento del intercambio bancario, luego de las 3:00 PM (hora local de Puerto Rico). 

<Note>Cada comercio puede definir su ventana de corte (generando transacciones con un Batch/Lote nuevo a la hora de corte establecida). Las transacciones que van a entrar a la reconciliación son las que correspondan al Batch/Lote de esa ventana de procesamiento.</Note>

## Reconciliación Interna

Una vez se realiza el cierre de la red procesadora, se procede a hacer una reconciliación interna entre el motor de pago y el depósito bancario (settlement). 

Esta reconciliación es realizada a través de un sistema automatizado de Evertec, el cual se encarga de verificar y validar que las transacciones hayan sido procesadas correctamente conforme a los registros establecidos.

## Depósito Bancario (Settlement)

Los depósitos bancarios se acreditan en la cuenta del Comercio el siguiente día hábil posterior al procesamiento de las transacciones y al cierre de la red procesadora.

Es importante señalar que, en el caso de American Express (AMEX), los plazos de depósito pueden variar según los términos y condiciones previamente negociados entre el cliente y dicha entidad.

<Note>Esta disposición no aplica para cuentas bajo el programa AMEX OptBlue.</Note>

## Portal Web Evertec
Es una plataforma en línea diseñada para que los comercios puedan consultar y gestionar información clave de sus operaciones electrónicas.

## ¿Cómo el Comercio hace su reconciliación?

Los comercios disponen de dos alternativas para llevar a cabo sus procesos de conciliación: mediante herramientas automatizadas o de forma manual.

En el caso de las soluciones automatizadas, los comercios pueden desarrollar sus propios programas o utilizar sistemas existentes que sean compatibles con la generación y lectura de reportes automatizados, tales como los archivos logfile y lockbox.

Por otro lado, si el proceso se realiza manualmente, el comercio puede utilizar el reporte operacional en formato PDF para verificar transacción por transacción. Alternativamente, si cuenta con acceso a la herramienta Predictive, también puede emplearla como apoyo en este proceso.

## Beneficios de un Proceso de Reconciliación

•	**Verifique transacciones:** le permite identificar si hay algún error en la transacción, como cargos duplicados, depósitos no registrados o errores de transcripción. 
 
•	**Posteo de pagos a sus clientes:** valide los pagos que se ejecutaron correctamente y actualice los balances o saldos de las cuentas de los clientes.
 
•	**Coteje saldos:** puede asegurar que el saldo bancario coincida con el saldo contable.

•	**Prevenga fraudes:** le ayuda a detectar transacciones no autorizadas o sospechosas.

•	**Cumpla con requisitos fiscales y auditorías:** le sirve como evidencia para auditorías internas o externas, y para cumplir con requisitos fiscales. 

•	**Optimice el manejo de finanzas:** con datos financieros actualizados y precisos, le permite planificar pagos, inversiones y presupuestos con mayor eficacia.

## Predictive

Es una solución de **Business Intelligence (BI)** que permite visualizar, analizar y anticipar información clave del negocio, incluyendo datos extraídos de archivos PDF como los estados de cuenta bancarios.

## ¿Cómo solicitar Predictive?

Para realizar esta solicitud, deberá comunicarse con su oficial de relación, quien le proporcionará todos los detalles pertinentes y la orientación necesaria para completar el proceso.
