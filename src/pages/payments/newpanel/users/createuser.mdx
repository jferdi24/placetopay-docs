# Creación de usuario

La creación de un usuario inicia a partir de una invitación enviada por correo electrónico. Para completar el registro, el usuario debe dar clic al botón Aceptar invitación incluido en el correo.  {{ className: 'lead' }}

<CenteredContainer>
   ![aceptarinvitacion](/payments/newpanel/migration/newuser/aceptarinvitacion.webp)
</CenteredContainer>

Al dar clic en el botón **Aceptar invitación** el sistema lo redirige al formulario de creación de cuenta, donde se van a solicitar los campos: 

- **Nombre**: Permite asignar nombre al usuario a crear. 

- **Correo electrónico**: Permite asignar un correo el cual quedara vinculado al usuario. 

- **Contraseña**: Permite configurar una contraseña, la cual permitirá el acceso a panel. 

- **Confirmación de contraseña**: Permite confirmar la contraseña ingresada en el campo anterior. 

<CenteredContainer>
![createuser](/payments/newpanel/migration/newuser/createuser.webp)
</CenteredContainer>

El sistema lo debe redirigir a la pantalla donde al correo electronico le llegara un codigo el cual debe de copiar en este apartado.

<CenteredContainer>
![confirmacionotp](/payments/newpanel/migration/newuser/confirmacionotp.webp)
</CenteredContainer>

Al completar la información del login de clic en la validación reCaptcha, de ser necesario cumpla con el reto para continuar, seguido de clic en el botón **Iniciar sesión**.  

## Doble factor de autenticación

A continuación se muestra la información correspondiente al doble factor de autenticación, el cual se debe cumplir para poder acceder al Panel, realizando los siguientes pasos: 

- Descargue una de las aplicaciones sugeridas en el paso 1 (Validar la siguiente imagen) 

- Escanee el codigo de Qr con la aplicación descargada. 

- Finalmente, ingrese el codigo generado en la aplicación tras escanear el codigo de barras. 

- De clic en el boton **Autenticar** para completar el proceso.  

Este proceso se describe en la interfaz del sistema de forma detallada: 

<CenteredContainer>
![2fa](/payments/newpanel/migration/newuser/2fa.webp)
</CenteredContainer>

Una vez la verificación del doble factor de autenticación sea confirmado con éxito se debe mostrar en pantalla los códigos de recuperación que se pueden usar en caso de que no pueda realizar con éxito el doble factor de autenticación usando las aplicaciones de autenticación.  

Se cuenta con el boton **Descargar**, el cual permite la descarga de los códigos, se recomienda guardarlos en un lugar seguro para que se puedan usar de ser necesario.   

<CenteredContainer>
![codigosderecuperacion](/payments/newpanel/migration/newuser/codigosderecuperacion.webp)
</CenteredContainer>

Al completar la autorización se debe abrir el **Panel** correspondiente al usuario creado, mostrando el Dashboard inicial.  

![homenewuser](/payments/newpanel/migration/newuser/homenewuser.webp)

## Login al panel

Para acceder al panel ingrese al enlace correspondiente a través de un navegador, se debe mostrar en pantalla los campos:
- Correo electrónico 
- Contraseña 

![loginnewpanel](/payments/newpanel/migration/newuser/loginnewpanel.webp)

Una vez completados los campos, de clic en el botón **Iniciar sesión** para acceder al sistema.  

Se debe mostrar en pantalla un mensaje solicitando el código de autenticación (Código de autenticación del 2FA) para poder acceder al sistema, seguido se debe contar con el campo para el ingreso de este.  

Se cuenta con la opción de marcar este dispositivo como uno de confianza, esto para que no se siga solicitando la autenticación al intentar acceder nuevamente desde el mismo dispositivo.  

Una vez ingrese el código otorgado en la aplicación de autenticación, de clic en el botón **Autenticar**. 

![login2fanewuser](/payments/newpanel/migration/newuser/login2fanewuser.webp)

Al completar la autenticación exitosa, el sistema permite acceder al **Panel**. 

![homenewuser](/payments/newpanel/migration/newuser/homenewuser.webp)

## Recuperar contraseña

<Note type="error"> Recomendamos verificar que las credenciales sean correctas e inténtarlo nuevamente, antes de restablecer la contraseña. </Note>

En caso de olvidar la contraseña, puede utilizar la opción **Olvidaste tu contraseña** para restablecerla.

![loginnewpanel](/payments/newpanel/migration/newuser/loginnewpanel.webp)

Donde digitaremos el correo electrónico y de clic en la validación reCaptcha, de ser necesario cumpla con el reto para continuar, seguido de clic en el botón **Enviar link**. 

<CenteredContainer>
![recuperacionpassword](/payments/newpanel/migration/newuser/recuperacionpassword.webp)
</CenteredContainer>

La pagina nos mostrará una confirmación en la cual nos indica que dentro de 5 minutos recibiremos el correo para restablecer nuestra contraseña.

![confirmacionrecuperacionpass](/payments/newpanel/migration/newuser/confirmacionrecuperacionpass.webp)
