# Dash

## Migración usuario desde Panel a Dash

Este documento describe el proceso de migración de cuentas de usuario desde el sistema Panel hacia la nueva versión del sistema, en adelante denominado Dash.  {{ className: 'lead' }}

## Requisitos previos

<Note type="error"> Contar con un usuario y contraseña activos en el sistema Panel.</Note>

<Note type="error"> Acceso al correo electrónico asociado a la cuenta para validaciones posteriores.</Note>

## Paso 1. Acceso a Dash

Inicie accediendo a la url correspondiente de Dash, donde se mostrará la interfaz para inicio de sesión con los campos correo electrónico y contraseña, además del botón Iniciar sesión y la opción que permite restablecer contraseña.

A continuación, se presenta las url para clientes LATAM:

| Cliente/País    | Panel             |  Dash          |   Tipo de Ambiente   |
|---------| -------------------------- | ------------------------------------------------------- | ---------------- |
| Panamá, Costa Rica, Puerto Rico, Colombia  | [Panel Placetopay LATAM](https://panel.placetopay.com/)     | [Placetopay Dash](https://dash.placetopay.com/login)  | Producción |
| Ecuador       | [Panel Placetopay EC](https://panel.placetopay.ec/)    | [Placetopay Dash](https://dash.placetopay.ec/login)     | Producción |
| AvalPay       | [Panel AvalPayCenter](https://admin.test.avalpaycenter.com/)    | [AvalPayCenter Dash](https://dash.avalpaycenter.com/login)    | Producción |
| Uruguay       | [Panel Placetopay UY](https://panel.placetopay.uy/)    | [PlacetoPay Dash UY](https://dash.placetopay.uy/login)    | Producción |

## Paso 2. Iniciar sesión

En el campo correo electrónico y contraseña debe ingresar los datos del usuario y contraseña con los que solía acceder a Panel, seguido de clic en **Iniciar sesión**.

![Dashome](/payments/newpanel/migration/newuser/loginhomees.webp)

## Paso 3. Validación de correo electrónico

El sistema lo direccionará a una pantalla donde se solicita el ingreso del correo electrónico al cual va a vincular el usuario, ingrese el correo electrónico y de clic en **validar email.**

![Migration](/payments/newpanel/migration/newuser/validatemailes.webp)

## Paso 4. Validación de datos

Al ingresar y validar el correo electrónico se van a desplegar los campos:

- Nombre: Permite ingresar el nombre con el que va a quedar el usuario en el Dash.

- Contraseña: Permite crear una contraseña de acceso, teniendo en cuenta los criterios sugeridos.

Además, se cuenta con los campos regresar y siguiente.

De clic en **Siguiente** para continuar con el proceso.

![nextsteps](/payments/newpanel/migration/newuser/confirmvalidationes.webp)

## Paso 5. Confirmación de correo electrónico

El sistema le solicitará ingresar el código enviado al correo, esto con el fin de confirmar el correo electrónico vinculado a la cuenta. Además, se presentan los botones **Regresar** y **Validar código**.

Una vez reciba el correo con el código, ingréselo y de clic en **Validar código**.

![otp](/payments/newpanel/migration/newuser/codeotpes.webp)

En pantalla se mostrará un mensaje indicando que la transferencia fue exitosa, se indica el correo electrónico validado y el botón **Iniciar sesión**.

![sucess](/payments/newpanel/migration/newuser/confirmvalidationotpes.webp)

## Paso 6. Finalización del proceso de migración

Una vez de clic en **Iniciar sesión**, se inicia el acceso a Dash, y se presentará el mensaje: **El proceso de migración puede tardar unos minutos**

Al finalizar el proceso de migración se visualizarán las opciones en Dash según los permisos con los que cuente el usuario migrado.

![Dash](/payments/newpanel/migration/newuser/dashes.webp)

## Acceso a Dash tras migración de usuario

Para acceder a Dash por primera vez, tras haber realizado la migración del usuario, debe seguir los siguientes pasos, con el fin de completar la validación de doble factor de autenticación sugerida por el sistema:

## Paso 1. Acceder a Dash

Para acceder a Dash ingrese al enlace correspondiente a través de un navegador. Se debe mostrar en pantalla los campos Correo electrónico y contraseña.

![login](/payments/newpanel/migration/newuser/loginhomees.webp)

Ingrese el correo y la contraseña que definió tras el proceso de migración de su cuenta, seguido de clic en el botón **Iniciar sesión** para acceder al sistema.

## Paso 2. Confirmación doble factor de autenticación

A continuación se muestra la información correspondiente al doble factor de autenticación, el cual se debe cumplir para poder acceder al Dash, realizando los siguientes pasos:

- Descargue una de las aplicaciones sugeridas en el paso 1 (Validar la siguiente imagen)

- Escanee el código QR con la aplicación descargada.

- Finalmente, ingrese el codigo generado en la aplicación tras escanear el codigo de barras.

- De clic en el boton Autenticar para completar el proceso.

<CenteredContainer>
![qr](/payments/newpanel/migration/newuser/2FAqres.webp)
</CenteredContainer>

Una vez la verificación del doble factor de autenticación sea confirmada con éxito se debe mostrar en pantalla los codigos de recuperación, los cuales deben ser descargados para poder continuar con el proceso. De clic en el botón **Descargar**.

## Paso 3. Descargar los codigos de recuperación

Al realizar la descarga de los códigos de recuperación, se habilita el botón **Continuar**, de clic.

<CenteredContainer>
![recuperateotp](/payments/newpanel/migration/newuser/codigosderecuperacion.webp)
</CenteredContainer>

## Paso 4. Ingreso al Dash

Al completar el proceso se debe abrir el Dash correspondiente al usuario creado, mostrando el Dashboard inicial.

![Dashhome](/payments/newpanel/migration/newuser/dashes.webp)
