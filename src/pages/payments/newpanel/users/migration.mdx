# Nuevo Panel
La siguiente documentación tiene como alcance ilustrar los pasos necesarios para lograr el proceso de migración de cuenta de usuario del sistema Viejo Panel hacia la nueva versión del sistema, en adelante descrita como Nuevo Panel. 

## Migración usuario desde Viejo Panel a Nuevo Panel 

Para realizar esta migración es necesario contar con un usuario y contraseña activo que permita el acceso a Viejo Panel.  

1. Inicie accediendo a la url correspondiente de Nuevo Panel, donde se mostrará la interfaz para inicio de sesión con los campos correo electrónico y contraseña, además del botón Iniciar sesión y la opción que permite restablecer contraseña.  

    A continuación, se presenta las url para clientes Latam, Colombia  

    | Cliente/País    | Viejo Panel             |  Nuevo Panel          |   Tipo de Ambiente   |
    |---------| -------------------------- | ------------------------------------------------------- | ---------------- |
    | LATAM, Colombia  | [Panel Placetopay LATAM](https://panel.placetopay.com/)     | [Placetopay Dash](https://dash.placetopay.com/login)  | Producción | 
    | Ecuador       | [Panel Placetopay EC](https://panel.placetopay.ec/)    | [Placetopay Dash](https://dash.placetopay.ec/login)     | Producción | 
    | AvalPay       | [Panel AvalPayCenter](https://admin.test.avalpaycenter.com/)    | [AvalPayCenter Dash](https://dash-test-gou.placetopay.com/login)    | Test | 
    | Uruguay       | [Panel Placetopay UY](https://panel.placetopay.uy/)    | [PlacetoPay Dash UY](https://dash.placetopay.uy/login)    | Producción | 

2. En el campo correo electrónico y contraseña debe ingresar los datos del usuario y contraseña con los que solía acceder a Viejo Panel, seguido de clic en **Iniciar sesión**.  

![Dashome](/newpanel/migration/imagen1.webp)

3. El sistema lo direccionará a una pantalla donde se solicita el ingreso del correo electrónico al cual va a vincular el usuario, ingrese el correo electrónico y de clic en validar email.  

![Migration](/newpanel/migration/imagen2.webp)

4. Al ingresar y validar el correo electrónico se van a desplegar los campos: 

- Nombre: Permite ingresar el nombre con el que va a quedar el usuario en el Nuevo Panel. 

- Contraseña: Permite crear una contraseña de acceso, teniendo en cuenta los criterios sugeridos.  

Además, se cuenta con los campos regresar y siguiente.  

5. De clic en **Siguiente** para continuar con el proceso.  

![nextsteps](/newpanel/migration/imagen3.webp)

6. El sistema le solicitará ingresar el código enviado al correo, esto con el fin de confirmar el correo electrónico vinculado a la cuenta. Además, se presentan los botones **Regresar** y **Validar código**.  

Una vez reciba el correo con el código, ingréselo y de clic en **Validar código**.  

![otp](/newpanel/migration/imagen4.webp)

7. En pantalla se mostrará un mensaje indicando que la transferencia fue exitosa, se indica el correo electrónico validado y el botón **Iniciar sesión**.

![sucess](/newpanel/migration/imagen5.webp)

8. Una vez de clic en **Iniciar sesión**, se inicia el acceso a Nuevo Panel, y se presentará el mensaje: **El proceso de migración puede tardar unos minutos**

9. Al finalizar el proceso de migración se visualizarán las opciones en Nuevo Panel según los permisos con los que cuente el usuario migrado.  

![Dash](/newpanel/migration/imagen6.webp)

## Acceso a Nuevo Panel tras migración de usuario

Para acceder a Nuevo Panel por primera vez, tras haber realizado la migración del usuario, debe seguir los siguientes pasos, con el fin de completar la validación de doble factor de autenticación sugerida por el sistema:  

1. Para acceder a Nuevo Panel ingrese al enlace correspondiente a través de un navegador. Se debe mostrar en pantalla los campos Correo electrónico y contraseña.  

![login](/newpanel/migration/imagen7.webp)

2. Ingrese el correo y la contraseña que definió tras el proceso de migración de su cuenta, seguido de clic en el botón **Iniciar sesión** para acceder al sistema.  

3. A continuación se muestra la información correspondiente al doble factor de autenticación, el cual se debe cumplir para poder acceder al panel, realizando los siguientes pasos: 

- Descargue una de las aplicaciones sugeridas en el paso 1 (Validar la siguiente imagen) 

- Escanee el código QR con la aplicación descargada. 

- Finalmente, ingrese el codigo generado en la aplicación tras escanear el codigo de barras. 

- De clic en el boton Autenticar para completar el proceso.

![qr](/newpanel/migration/imagen8.webp)

![finishqr](/newpanel/migration/imagen9.webp)

4. Una vez la verificación del doble factor de autenticación sea confirmada con éxito se debe mostrar en pantalla los codigos de recuperación, los cuales deben ser descargados para poder continuar con el proceso. De clic en el botón **Descargar**.  

5. Al realizar la descarga de los códigos de recuperación, se habilita el botón **Continuar**, de clic. 

![recuperateotp](/newpanel/migration/imagen10.webp)

6. Al completar el proceso se debe abrir el Nuevo Panel correspondiente al usuario creado, mostrando el Dashboard inicial.  

![Dashhome](/newpanel/migration/imagen11.webp)
