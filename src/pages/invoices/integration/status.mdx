# Estados de la orden de pago

La orden de pago puede tener los siguientes estados posibles:

| **Estado** | **Significado** |
|------------|-----------------|
| `PAYED`    | Pagada          |
| `EXPIRED`  | Expirada        |
| `ACTIVE`   | Activa          |
| `INACTIVE` | Inactiva        |
| `HOLD`     | Bloqueada       |
