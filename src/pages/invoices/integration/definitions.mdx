# Definiciones clave

## Ordenes de pago - ( Factura ) - _**Invoice**_

Es el documento digital emitido por el sistema de facturación. Cada orden de pago cuenta con un
**id único** asignado automáticamente, el cual es utilizado en todas las operaciones de
bloqueo, liberación, eliminación o consulta.

Una orden de pago puede encontrarse en distintos estados (`ACTIVE`, `BLOCKED`, `EXPIRED`, etc.)
y contiene información del deudor, valores, impuestos y fechas clave.

---

## Bloqueo de orden de pago - _**Invoice hold**_

Es el proceso mediante el cual una orden de pago se **marca como bloqueada**.
Cuando está bloqueada, la orden de pago presenta un estado de pago **“en proceso”** en el panel
administrativo y **no puede ser editada ni eliminada**.
El bloqueo se revierte mediante la **liberación de orden de pago**.

---

## Liberación de orden de pago - _**Invoice Release**_

Es la acción opuesta al bloqueo. Una orden de pago liberada queda nuevamente activa y disponible
para operaciones de edición o eliminación, siempre y cuando no esté pagada.

---

## Eliminación de orden de pago - _**Invoice remove**_

Proceso que elimina de manera **definitiva** una orden de pago individual del sistema.
La operación es **irreversible** y, aunque la orden de pago se elimina, la respuesta del servicio
devuelve la información que registraba antes de ser eliminada, como evidencia de la operación.

---

## Eliminación masiva - _**Invoice flush**_

Proceso que elimina **todas las ordenes de pago** de un sitio que no esten **pagadas o bloqueadas**.
La acción es irreversible y está pensada para procesos de limpieza masiva del sistema.

---


## Fechas de una orden de pago - _**Invoice dates**_

Una orden de pago incluye tres fechas clave para su gestión:

- **createdAt** → Fecha en que fue creada.
- **expirationDate** → Fecha de vencimiento.
- **finalDate** → Fecha máxima para la liquidación o uso.

---

## Deudor - _**Debtor**_

Entidad o persona asociada a una orden de pago, que representa al responsable del pago.
Incluye información básica como número de documento, tipo, nombre y apellido.
