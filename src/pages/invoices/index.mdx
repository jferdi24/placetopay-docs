Este servicio web y su arquitectura permite a cualquier aplicación conectarse a Placetopay para consumir los servicios de facturas de Micrositios.

Para usar este servicio web, su aplicación deberá tener un lenguaje de programación que pueda comunicarse con un servicio de tipo REST.

El método de integración contempla las operaciones básicas para creación, obtención, asentamiento y remoción de facturas.

A continuación, se describirán para cada uno de los endpoints, los datos requeridos para la petición, así como los datos resultantes en la respuesta y ejemplos de peticiones y respuestas para cada uno de los casos
